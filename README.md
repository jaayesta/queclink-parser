> Parses raw data from Queclink devices (TCP). Devices supported: GV55, GV200, GV300, GV75W, GV300W, GV500, GV600W GV800W, GMT100, GL50, GL50B, GL300, GL300W

## Installation

```bash
npm i -S queclink-parser
```
